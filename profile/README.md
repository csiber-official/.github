# CSIBER Official · Open Source @ CSIBER Kolhapur

[![CSIBER Logo](https://csiber.ac.in/wp-content/uploads/2023/08/CSIBER-Logo.png)](https://csiber.ac.in)

**Chhatrapati Shahu Institute of Business Education & Research**  
Autonomous | NAAC A+ | Shivaji University, Kolhapur

This is the **official open-source organization** of CSIBER students, faculty, alumni and tech clubs.

Every repository automatically gets its own live website at  
`https://csiber-official.github.io/repository-name` → [Visit our portal](https://csiber-official.github.io)

## Our Tech Clubs & Labs
- **Coding Club** → Competitive programming & hackathons  
- **AI/ML Lab** → Machine Learning, Data Science projects  
- **CyberSec Crew** → Cybersecurity, CTFs, ethical hacking  
- **WebDev Society** → Full-stack web projects, WordPress, React  
- (more clubs coming every semester!)

## How to get involved
1. Fork any repo → Make changes → Open a Pull Request  
2. Want your own project site? Just create a repo under this org (Pages auto-enabled)  
3. Join our WhatsApp/Discord community (links inside club repos)
